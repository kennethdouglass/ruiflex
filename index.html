Responsive UX with Flex

CSS Flexbox allows us to create a responsive UX that is responsive to internal elements rather than the viewport width using Media Queries. It is far more easier to implement IMHO. It may not always provide the exact behavior we desire or expect but I can see where there could be many applications of its use, especially if we combine its use with media queries.

This demo creates a grid layout. Each card container sets the values of flex-grow and flex-shrink to 1, which allows the card to grow and shrink to fill out the width of the parent element. The value of flex-basis is the key. In this demo, each card container's width is determined by the value of flex-basis, in this case 200 pixels. This defines the default size of the container before the remaining space is distributed. The parent container has a max-width of 820px, which provides space for 4 cards in each row. In the last row there are only 2 cards, which grow to fill the allowed space. If the viewport is narrowed below the max-width, the number of cards on a row adjusts and the cards in the row grow to fill the space.
